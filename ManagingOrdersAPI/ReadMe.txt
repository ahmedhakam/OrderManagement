Database is initially empty

Add order using [post] orders in swagger page

Execute [GET] orders to get all orders 

Execute [GET] orders/id to get a single order by id

Execute [PUT] order/id to update order status by id

Execute [Delete] order/id to delete single order by id

Execute [Get] orders/searchCustomer to search for orders made by a specific customer using his name

Execute [Get] orders/searchStatus to get all orders with a specific order status
